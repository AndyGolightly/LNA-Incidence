R code to implement the simulation study.

mainLNA.r - implements the forward-filter based Metropolis-Hastings scheme
mainMJP.r - implements pseudo-marginal MH for an MJP model of cumulative incidence
mainCPMMH.r - implements (correlated) pseudo-marginal MH with the LNA as the latent incidence model
